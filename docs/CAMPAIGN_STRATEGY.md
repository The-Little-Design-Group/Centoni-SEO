## Tracking Requirements

### Conversion Actions to Configure

1. **Form Submissions**
   - Contact form completions
   - Consultation request forms
   - Quote request forms

2. **Phone Calls**
   - Calls from ads (call extensions)
   - Calls from website (call tracking number)
   - Duration threshold: 60 seconds minimum

3. **Engagement Events**
   - Portfolio page views (secondary conversion)
   - Time on site over 2 minutes (secondary conversion)

### Analytics Integration

- Link Google Ads to Google Analytics 4
- Enable auto-tagging
- Import GA4 conversions to Google Ads
- Set up audience segments for remarketing

## Budget Framework

### Monthly Budget Allocation (Example: $5,000/month)

| Campaign | Allocation | Monthly Budget |
|----------|------------|----------------|
| Authority/Brand | 40% | $2,000 |
| Historic Homes | 30% | $1,500 |
| Investor/Value-Add | 20% | $1,000 |
| Partnerships | 10% | $500 |

### Bid Guidance

| Ad Group Type | Expected CPC Range | Notes |
|---------------|-------------------|-------|
| Kitchen/Bath | $15-35 | High competition |
| Whole Home | $20-40 | High-value leads |
| Historic | $10-25 | Niche, lower competition |
| Seismic | $8-20 | Specialized, moderate competition |
| Investor | $12-30 | Variable, test-dependent |

**Note:** San Francisco is a premium market. Expect higher CPCs than national averages.

## Performance Benchmarks

### Target Metrics (After Learning Period)

| Metric | Target |
|--------|--------|
| CTR (Search) | Greater than 5% |
| Quality Score (Avg) | Greater than 7 |
| Conversion Rate | Greater than 3% |
| Cost Per Lead | Less than $150 |
| ROAS | Greater than 5:1 (long-term) |

### Optimization Schedule

| Task | Frequency |
|------|-----------|
| Search terms review | Weekly |
| Bid adjustments | Weekly |
| Ad copy testing | Bi-weekly |
| Negative keyword updates | Weekly |
| Landing page analysis | Monthly |
| Campaign restructuring | Quarterly |
